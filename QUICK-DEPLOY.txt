📋 ULTRA-QUICK 3-STEP DEPLOYMENT
═══════════════════════════════════════════════════════════════════════════════

STEP 1: BUILD
─────────────────────────────────────────────────────────────────────────────
Open Terminal/Command Prompt in project folder:

  npm install
  npm run build

✅ Creates 'dist' folder with your website
⏱️ Takes 2-3 minutes

STEP 2: UPLOAD TO AFRIHOST
─────────────────────────────────────────────────────────────────────────────
1. Login: https://clientzone.afrihost.com
2. Click: cPanel → File Manager
3. Go to: public_html
4. Delete: All existing files (if any)
5. Upload: CONTENTS of 'dist' folder
   - index.html
   - .htaccess
   - assets/ folder (with everything inside)

✅ Files uploaded to public_html (NOT in a subfolder!)
⏱️ Takes 5-10 minutes

STEP 3: TEST
─────────────────────────────────────────────────────────────────────────────
Visit: https://dentxquarters.co.za

✅ Website should load!
⏱️ Takes 30 seconds

📁 CORRECT FILE STRUCTURE AFTER UPLOAD
═══════════════════════════════════════════════════════════════════════════════

✅ CORRECT:
public_html/
├── .htaccess
├── index.html
└── assets/
    ├── index-abc123.css
    ├── index-xyz789.js
    └── ...

❌ WRONG:
public_html/
└── dist/                    ← Don't upload dist folder itself!
    ├── index.html
    └── assets/

❌ WRONG:
public_html/
├── src/                     ← Don't upload source code!
├── node_modules/
└── package.json

🆘 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

BLANK PAGE?
→ Check files are in public_html (not in dist subfolder)
→ Clear browser cache (Ctrl+F5)
→ Read: FIX-BLANK-WEBSITE.md

404 ERROR?
→ Check .htaccess file exists in public_html
→ Enable "Show Hidden Files" in File Manager
→ Contact Afrihost to enable mod_rewrite

BOOKING FORM NOT WORKING?
→ Verify .htaccess uploaded
→ Check routing rules in .htaccess
→ Test on different browsers
